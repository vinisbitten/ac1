### Exercício 01

- Faça uma função que efetua o cálculo do salário líquido de um professor. Os dados fornecidos serão:
  - valor hora aula
  - número de aulas dadas
  - percentual de desconto do INSS.
- [Código](./ex1/salario.py)
- [Testes](./ex1/test_salario.py)
